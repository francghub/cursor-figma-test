### Image Specifications:

- **Format:** JPG, PNG, or WebP
- **Recommended Size:** 1920x1080px or larger
- **Aspect Ratio:** 16:9 (landscape)
- **File Size:** Optimize to < 500KB per image for better performance
- **Quality:** High quality, suitable for full-screen backgrounds

### Optimization Tips:

- Use tools like [TinyPNG](https://tinypng.com/) or [Squoosh](https://squoosh.app/) to compress images
- Convert to WebP format for best performance (optional)
- Ensure images look good even when darkened (parallax overlay effect)
